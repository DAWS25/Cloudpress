# ☁️ CloudPress

CloudPress é um projeto de **laboratório e aprendizado contínuo** focado em **AWS, Cloud Computing e boas práticas de SRE/DevOps**.  
Ele nasce como um ambiente prático para estudos, testes e validações de arquiteturas em nuvem.

O objetivo é criar uma **plataforma de gerenciamento de conteúdo**, capaz de organizar e entregar diferentes formatos de informação, como:

- 📄 Artigos técnicos  
- 🎥 Vídeos  
- 🖼️ Imagens  
- 📝 Postagens e materiais educacionais  

Tudo isso servirá como base para experimentos reais de arquitetura, automação, segurança, observabilidade e escalabilidade em cloud.

---

## 🎯 Objetivos do Projeto

- Servir como **laboratório prático de AWS**
- Aplicar conceitos estudados no curso **Descomplicando AWS**
- Explorar arquiteturas modernas e escaláveis
- Praticar **Infraestrutura como Código (IaC)**
- Implementar pipelines de **CI/CD**
- Trabalhar conceitos de **SRE, DevOps e FinOps**
- Criar uma base real para evolução técnica contínua

---

## 🧪 Escopo Inicial

O projeto CloudPress será evoluído de forma incremental, incluindo:

- Provisionamento de infraestrutura na AWS
- Deploy de aplicações em ambientes controlados
- Gerenciamento de conteúdo (CRUD)
- Autenticação e autorização
- Observabilidade (logs, métricas e alertas)
- Segurança e boas práticas
- Otimização de custos

### Aplicação
- Backend (a definir)
- Frontend (React)
- APIs REST

### 🔐 Autenticação
O Cloudpress utiliza Amazon Cognito para gerenciamento de identidade, garantindo segurança, escalabilidade e integração nativa com serviços AWS.
A infraestrutura é provisionada via CloudFormation, seguindo boas práticas de infraestrutura como código.


> ⚠️ As tecnologias podem mudar conforme a evolução do laboratório e dos estudos.
